--------------------------------------------------------------------------------      
* Enable 2 GPIO as OutPut.
* Active Label Of GPIO Name As CLK,DATA .
* Add Tm1637 library to your project.
* Configure `TM1637.h` , `TM1637.c` files.
* Call `TM1637_Init(); ` in infinit main.
